# y-draft-jjs

developing
1. 启动websoket-server
```
npm run start:server
```
2. 启动demo
```
npm run dev
```

性能数据:

更新频率 ≥  1次/70ms  光标位置更改错误率   ≤ 1%

更新频率 ≈= 1次/36ms  光标位置更改错误率  (用户变更光标位置频率 = 1次/700ms 时 为 2%， <= 1次/300ms 为 5%))

更新频率 ≤ 1次/24ms  光标位置更改错误率 (用户变更光标位置频率 = 1次/700ms 时 为 4%， <= 1次/300ms 为 5%)
