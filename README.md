# y-draft-jjs

developing

1. 启动 websoket-server

```
npm run start:server
```

2. 启动 demo

```
npm run dev
```

性能数据:

更新频率 ≥ 1 次/70ms 光标位置更改错误率 ≤ 1%

更新频率 ≈= 1 次/36ms 光标位置更改错误率 (用户变更光标位置频率 = 1 次/700ms 时 为 2%， <= 1 次/300ms 为 5%))

更新频率 ≤ 1 次/24ms 光标位置更改错误率 (用户变更光标位置频率 = 1 次/700ms 时 为 4%， <= 1 次/300ms 为 5%)
